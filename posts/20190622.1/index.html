一天一个计算机词: 框架

2019-06-22
#中国語学習  #计算机词汇  #IT用語 

框架 kuàngjiàとは、英語ではFramework、日本語ではフレームワークのことだ。たとえば、Ruby on RailsなどのWebアプリケーションフレームワークは、框架の一つである。なお、WebアプリケーションフレームワークはWeb 应用框架 yìngyòng kuàngjiàと呼ばれる。

框架の通常の意味は、構造や枠組み、仕組みなどである。ここから派生して、アプリケーションを作成するための一般化された構造に対してこの語が用いられるようになったのだろう。

使用例:

汉语 拼音 日语
Web 应用框架是创建 web 应用的一种方式。 Web yìngyòng kuàngjià shì chuàngjiàn web yìngyòng de yì zhǒng fāngshì. Webアプリケーションフレームワークは、ウェブアプリケーションを作成するための一方式である。
Next.js 是一个流行的、轻量级的框架，用于配合 React 打造静态化和服务端渲染应用。 Next.js shì yíge liúxíng de, qīng liàng jí de kuàngjià, yòng yú pèihé React dǎzào jìngtài huà hé fúwù duàn xuànrǎn yìngyòng. Next.jsは人気のある軽量なフレームワークで、Reactを用いて静的かつサーバーサイドでレンダリングされるアプリケーションを作成するために使用される。

例文中の他の重要な単語は以下である:

  • 轻量级的框架 qīng liàng jí de kuàngjià: lightweight framework
  • 静态化 jìngtài huà: static
  • 服务端渲染 fúwù duàn xuànrǎn: server-side rendering

最後の例はReactのドキュメンテーションから引用した。
